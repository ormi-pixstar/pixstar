# Pixstar

## 1. 목표와 기능
 ### 1.1 목표
  - 사용자들 간의 사진과 짧은 텍스트를 활용한 소통을 위한 공간을 만드는 것을 목적으로 한 게시판 서비스

 ### 1.2 기능
  - 사용자 관련 기능
   - 회원가입, 로그인, 로그아웃, 회원탈퇴, 회원정보수정

  - 게시물 관련 기능
   - 텍스트와 이미지가 삽입이 가능한 게시물 작성, 조회, 수정, 삭제

  - 댓글 관련 기능
   - 게시글 하단에 게시글을 조회한 사람들의 댓글 작성, 조회, 수정 삭제
   - 대댓글 작성 가능
